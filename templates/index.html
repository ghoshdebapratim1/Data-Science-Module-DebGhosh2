What does the project do?

This project aims to detect counterfeit banknotes using a dataset of numerical attributes such as variance and skewness of images. The project uses XGBoost, a popular machine learning algorithm, for classification. The dataset is divided into training and testing sets and cross-validated to choose the best hyperparameters for the XGBoost model. The model is then trained on the training set and tested on the testing set to evaluate its performance. Finally, the confusion matrices for the training and testing sets are displayed visually using the Plotly library, providing a clear representation of the model's predictions. The project's main goal is to provide an automated and reliable way of detecting counterfeit banknotes, enhancing security and reducing the risk of financial fraud.

Exploration of Data: Univariate Plots

The box plot is an effective way to visualize the distribution of numerical variables. For this dataset, we created 4 box plots, one for each of the numerical variables 'Image.Var', 'Image.Skew', 'Image.Curt', and 'Entropy'. Each box plot is split by the target variable class ('Bank Note' and 'Other'), allowing us to quickly compare the distribution of each variable for the two classes. We can see that for some variables, such as 'Image.Var' and 'Image.Skew', there is a noticeable difference in the distribution between the two classes, while for others, such as 'Entropy' and 'Image.Curt', the distributions are quite similar. Additionally, we created a bar chart for the target variable, which displays the number of instances in each class ('Bank Note' and 'Other'). This helps us understand the class distribution of the dataset and whether it is imbalanced or balanced. In this case, we can see that the number of instances in each class is relatively balanced, which is good for building a classification model.

Exploration of Data: Bivariate Heatmap

Visualizing the Correlation between Numerical Variables using Bivariate Heatmap

A bivariate heatmap is a type of heatmap that displays the relationship between two variables. In this project, a bivariate heatmap is created to show the correlation between each pair of numerical variables in the dataset. The heatmap uses color to represent the strength of the correlation between two variables, with deep blue indicating a positive correlation and light blue indicating a negative correlation. This type of visualization is useful for identifying which pairs of variables are strongly correlated, which can help in feature selection for machine learning models. The bivariate heatmap can also give an idea of which pairs of variables might be useful for creating new features.

Exploration of Data: Confusion Matrix Heatmap

We've trained and tested the model, generating the following heatmap.

The confusion matrix heatmap is a powerful tool for visualizing the performance of a machine learning model. It displays the number of correct and incorrect predictions made by the model for each class in a tabular format. The diagonal cells of the matrix represent the number of correct predictions, while the off-diagonal cells represent incorrect predictions. The heatmap provides a color-coded representation of the matrix, with colors ranging from lighter to darker shades, indicating the number of predictions in each cell. This allows for a quick and easy interpretation of the model's accuracy and misclassification rates. By exploring the confusion matrix heatmap, we can gain valuable insights into the performance of our model, identify areas for improvement, and make data-driven decisions to optimize its performance.


Machine Learning


XGBoost Classifier: Explanation

XGBoost is a popular machine learning algorithm that helps us make predictions based on data. Imagine you have a set of images, some of which show real banknotes and some of which show fake banknotes. Your task is to teach the machine learning model how to tell the difference between real and fake banknotes based on features like the variance and skewness of the images. XGBoost helps you do this by using a mathematical model that analyzes these features and predicts whether a banknote is real or fake. The algorithm uses a process called gradient boosting to improve its accuracy over time, by making small adjustments to its predictions based on the mistakes it makes. With XGBoost, you can train a computer to identify real and fake banknotes quickly and accurately, helping to prevent financial fraud and improve security.

Conclusion


In conclusion, our project aimed to detect banknotes using numerical attributes of the images such as variance and skew. We used the XGBoost algorithm to classify the banknotes and achieved an accuracy of 99.1% on the testing dataset. Additionally, the confusion matrix heatmaps showed that our model had very few misclassifications. Our project's success indicates that numerical attributes of images can be used to classify banknotes accurately. This technique can be extended to other domains where numerical attributes of images can be used to classify and detect different objects.
